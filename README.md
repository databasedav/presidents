# presidents

## Requirements
- Python 3.7.0

## Development Setup
In your local `presidents` directory:
```
# installs required packages from pip and npm
npm run setup
source venv/bin/activate
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
